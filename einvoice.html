E-Invoice (IRN)

Mandatory for B2B supplies if AATO > ₹ 5 Cr · 64-char hash from IRP · auto-populates GSTR-1 · 30-day generation window · 24-hour cancel window

+ New IRN
IRN vs EWB — the simple split: IRN is for the invoice itself. EWB is for the movement of goods. IRN is mandatory for all B2B > ₹ 5 Cr taxpayers, once generated it never expires, and only the supplier ever generates one per invoice. The 64-char IRN hash is derived from Supplier GSTIN + Doc no + Doc date + Doc type + FY — deterministic, so duplicate invoices are rejected at the portal. Once issued, the IRP signs the JSON, returns a QR code (to be printed on invoice), and auto-pushes the data into your GSTR-1.
