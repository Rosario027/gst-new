Validations applied on Run Validate
All checks below run automatically on the uploaded data.
✓
GSTIN format & structure
15-char structure, checksum digit, state code, PAN linkage. (Live GSTN status when API connected.)
✓
Place-of-supply consistency
Recipient GSTIN state must match PoS. Mismatch flagged; drives IGST vs CGST+SGST.
✓
Reverse-arithmetic check
Taxable × rate ≈ tax (±₹1). IGST and CGST/SGST cannot coexist; CGST = SGST.
✓
Invoice-number rules
≤16 chars, alphanumeric with - and / only. Duplicate check per supplier.
✓
HSN master validation
Valid HSN/SAC (chapter + 4/6/8-digit). 6-digit recommended (mandatory if AATO > ₹5 Cr).
✓
Round-off & cross-foot
Invoice grand total = taxable + tax + cess, round-off within ±0.99.
